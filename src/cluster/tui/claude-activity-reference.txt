Key observations:
- Text blocks start with ● (bullet)
- Tool use: ● ToolName(args) — bold
- Tool result: ⎿  summary — indented, dimmed
- Diff output indented under tool result with +/- line markers
- Each block separated by blank line
- Final text block is the assistant's summary
